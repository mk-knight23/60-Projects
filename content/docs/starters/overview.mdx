## ðŸ“¦ Quick Start

```bash
# Using a starter template
npx create-next-app@latest my-app --example=[starter-name]

# Or clone directly
git clone [starter-repo]
cd my-app
npm install
cp .env.example .env.local
# Add your API keys
npm run dev
```
