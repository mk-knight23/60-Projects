## üîß Setup & Run

```bash
# Clone any web app project
git clone [repository-url]

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Add your API keys
# DATABASE_URL=
# STRIPE_SECRET_KEY=
# OPENAI_API_KEY=

# Run database migrations (if applicable)
npm run db:migrate

# Start development server
npm run dev
```
