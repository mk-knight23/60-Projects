# Install Tools

*Video walkthrough coming soon*

Lets install our develoment tools outlined earlier:

## Mac

### 1. Install Homebrew (if you don't have it)

Download from [brew.sh](https://brew.sh) → Run the `.pkg` installer.

### 2. Install Git & GitHub CLI

```bash
brew install git gh
```

### 3. Authenticate with GitHub

```bash
gh auth login
```

Follow the prompts — it opens your browser, you log in, done.

### 4. Install Node.js

```bash
brew install node
```

### 5. Install AI Code Editor

**Cursor (Recommended):**

Download from [cursor.com](https://cursor.com) → Install → Sign in.

After installing:
- Settings → Models → Select **Claude Opus 4.5**
- Settings → Features → Enable "Always allow file edits"
- Settings → Files: Auto Save → Set to **afterDelay**

**Claude Code (Alternative):**

If you prefer working in the terminal, install Claude Code from [claude.ai/code](https://claude.ai/code) and follow their setup guide.

## Windows

### 1. Install Git

Download from [git-scm.com](https://git-scm.com/download/win) → Run installer (use defaults).

### 2. Install GitHub CLI

Download from [cli.github.com](https://cli.github.com) → Run installer.

### 3. Authenticate with GitHub

Open Command Prompt or PowerShell:

```bash
gh auth login
```

Follow the prompts — it opens your browser, you log in, done.

### 4. Install Node.js

Download from [nodejs.org](https://nodejs.org) → Click **LTS** → Run installer.

### 5. Install AI Code Editor

**Cursor (Recommended):**

Download from [cursor.com](https://cursor.com) → Install → Sign in.

After installing:
- Settings → Models → Select **Claude Opus 4.5**
- Settings → Features → Enable "Always allow file edits"
- Settings → Files: Auto Save → Set to **afterDelay**

**Claude Code (Alternative):**

If you prefer working in the terminal, install Claude Code from [claude.ai/code](https://claude.ai/code) and follow their setup guide.
