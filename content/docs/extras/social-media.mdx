# Social Media (getlate.dev)

Post to multiple social platforms with one API.

## What is getlate.dev?

[getlate.dev](https://getlate.dev) replaces 13 social media integrations with one API.

- Post to multiple platforms
- Schedule posts
- Get analytics
- One endpoint for everything

Supported platforms include Twitter/X, LinkedIn, Facebook, Instagram, and more.

## When You Need It

When your app needs to:
- Post to social media for users
- Schedule social content
- Pull social analytics

## Setup

### 1. Create Account

1. Go to [getlate.dev](https://getlate.dev)
2. Sign up
3. Connect your social accounts

### 2. Get API Key

1. Go to your dashboard
2. Create an API key
3. Add to `.env.local`: `GETLATE_API_KEY=your-key`

## Prompt Templates

### Add Social Posting

```
I want users to post to social media from my app.

Platforms: [LIST: e.g., "Twitter, LinkedIn"]

Please:
1. Create a utility in lib/ for getlate.dev
2. Create an API route for posting
3. Create a simple UI for composing posts
4. Handle errors gracefully

Use GETLATE_API_KEY from environment.
```

### Schedule Social Posts

```
I want users to schedule social media posts.

Please:
1. Add scheduling to my posting form
2. Send schedule time to getlate.dev API
3. Show scheduled posts in a list
4. Allow canceling scheduled posts

Use getlate.dev's scheduling feature.
```

### Social Analytics

```
I want to show users their social media analytics.

Please:
1. Fetch analytics from getlate.dev
2. Display engagement metrics
3. Show a simple chart or stats
4. Update periodically

Keep the UI simple with DaisyUI.
```

## Testing

1. Connect a test social account
2. Make a test post
3. Verify it appears on the platform
4. Check analytics after some time
