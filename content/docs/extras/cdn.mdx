# CDN (Bunny)

Serve images and files faster worldwide.

## What is a CDN?

A CDN (Content Delivery Network) stores your files on servers around the world. Users download from the nearest server = faster loading.

Use for:
- User-uploaded images
- Large files
- Static assets

## When You Need It

- Your app has lots of images
- Users upload files
- You serve files to users globally

**Don't need it yet?** Skip this — add it when you actually need it.

## Setup

### 1. Create Bunny Account

1. Go to [bunny.net](https://bunny.net)
2. Sign up (pay as you go, very cheap)

### 2. Create Storage Zone

1. Go to **Storage** → **Add Storage Zone**
2. Choose a name and region

### 3. Create Pull Zone

1. Go to **Pull Zones** → **Add Pull Zone**
2. Connect to your storage zone
3. Get your CDN URL
